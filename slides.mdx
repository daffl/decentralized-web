## Public-key Cryptography

```sh
# Generate private key
openssl genpkey -algorithm RSA -pkeyopt rsa_keygen_bits:2048
  -out private_key.pem.pem

# Generate public key
openssl rsa -pubout -in private_key.pem
  -out public_key.pem
```

# Solid

[Solid](https://solid.inrupt.com/) empowers users and organizations to separate their data from the applications that use it.
It allows people to look at the same data with different apps at the same time.
