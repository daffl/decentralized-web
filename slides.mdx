## The decentralized web

- Trust: Blockchain, Consensus Protocols
- Publishing: Distributed file systems
  - [DAT](https://datproject.org/), [Beaker Browser](https://beakerbrowser.com), [IPFS](https://ipfs.io/)
- Identity and ownership:
  - [Solid](https://solid.inrupt.com/)
  - Crypto wallets (e.g. [Brave browser](https://brave.com/), [Opera](https://www.opera.com/crypto))
  - Tokens and smart contracts  
- Payment: Cryptocurrencies
